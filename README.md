## How to play

The goal is to paint the whole map with your team's color by moving.

1. Open an issue with a team name as the title (cr/cg/cb)
2. Create a public repository named "gitland-client"
3. Create a file called "act" in your repository
4. Write a single command (up/down/left/right) to that file
5. Wait for the next turn (about 60 seconds)
6. Read the "log" file and the files inside "players/YOURNAME" for intel
7. Go back to step 4
